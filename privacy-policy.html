  • Personalization: Remember your preferences and provide tailored recommendations
  • Subscription Management: Process payments, manage subscriptions, enforce usage limits
  • Improve Services: Analyze usage patterns to enhance app functionality and user experience
  • Communication: Send technical notices, support messages, and important updates

3. Data Storage and Security

3.1 Data Storage

Your data is stored using the following services:

  • Supabase: User accounts, profiles, recipes, preferences (encrypted database)
  • Flutter Secure Storage: Authentication tokens on your device (encrypted)
  • Render: Backend API hosting (data in transit encrypted via HTTPS)

3.2 Security Measures

We implement appropriate technical and organizational measures to protect your personal information:

  • Encryption: Data encrypted in transit (HTTPS) and at rest
  • Access Control: Restricted access to personal data (authorized personnel only)
  • Secure Storage: Industry-standard cloud storage with security certifications
  • Regular Updates: Security audits, patches, and updates
  • Authentication: Secure token-based authentication with automatic refresh

However, no method of transmission over the Internet or electronic storage is 100% secure. While we strive to protect your data, we cannot guarantee absolute security.

When you subscribe, we collect:
